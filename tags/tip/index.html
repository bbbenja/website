Améliorer la complétion JavaScript d'IntelliJ IDEA

IntelliJ IDEA Tout le monde sait que nous adorons utiliser IntelliJ IDEA pour développer. Peu importe le langage IntelliJ est vraiment un excellent IDE qui nous permet d’être efficaces quasiment instantanément. Par contre, quand on arrive dans le monde du JavaScript on peut avoir quelques soucis pour être dans un environnement suffisamment typé où l’on sait ce qu’on va faire sans se poser trop de questions, ou tout simplement pouvoir bénéficier de l’aide de l’IDE pour savoir de quelles méthodes on dispose.
